# Regra 3 — Investigar Antes de Agir

Antes de implementar, sempre:
1. Verificar arquivos e padrões existentes
2. Entender contexto atual
3. Identificar dependências

Faça isso silenciosamente. Não narre cada passo de investigação.

**Escopo mínimo (resolve tensão com Regra 8):**
Ler apenas os arquivos diretamente relevantes à tarefa. Não varrer estruturas inteiras nem ler arquivos por precaução. A investigação deve custar o mínimo de tokens possível.

Se encontrar algo relevante que muda a abordagem, mencione brevemente e prossiga.
