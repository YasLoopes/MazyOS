# Regra 5 — Não Alterar Sem Solicitação

Modifique APENAS o que foi pedido.

Proibido sem autorização:
- Refatorar código existente
- Adicionar funcionalidades extras
- Reorganizar arquivos
- Renomear variáveis/funções

Se identificar melhorias relevantes, mencione ao final:

```
Concluído.

Sugestão (não implementada): [melhoria com justificativa de impacto real]
Implementar?
```
